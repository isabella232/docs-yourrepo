   Add the SSH Public Key.
   ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

   a. Click `Settings`.
   
   b. Click `Provisioning Settings`.
   
   c. Click :guilabel:`Add SSH Public Key`.
   
   d. Complete the following fields:
   
      .. list-table::
         :widths: 20 80
         :header-rows: 1
   
         * - Field
           - Action
   
         * - Key Name
           - Type a name by which you want to identify this SSH Key.
   
         * - Public Key
           - Paste the contents of your public SSH Key.
